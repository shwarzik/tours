
# Tours App

Live demo: [https://tours-test.netlify.app/](https://tours-test.netlify.app/)

## How to run locally

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.
